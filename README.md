# 太陽系行星探險網站 - 使用說明

## 項目概述

這是一個介紹八大行星的交互式網站，展示太陽系中每顆行星的信息。

## 項目結構

```
/
├── index.html              # 首頁
├── css/
│   └── style.css          # 主樣式表
├── js/
│   ├── main.js            # 主JavaScript文件
│   └── api.js             # API和數據管理
├── planets/               # 行星詳情頁面
│   ├── mercury.html       # 水星
│   ├── venus.html         # 金星
│   ├── earth.html         # 地球
│   ├── mars.html          # 火星
│   ├── jupiter.html       # 木星
│   ├── saturn.html        # 土星
│   ├── uranus.html        # 天王星
│   └── neptune.html       # 海王星
├── images/                # 圖像目錄（可選）
└── README.md              # 本文件

```

## 功能特性

### 首頁
- **Hero Image**: 展示壯觀的宇宙背景圖像
- **太陽系簡介**: 關於太陽系的文章介紹
- **太陽系圖**: 交互式的太陽系圖表
  - 顯示所有八個行星的軌道
  - 行星可點擊以導航到詳情頁面
  - 土星包含環系統的視覺表現
  - 懸停效果顯示行星名稱

### 行星詳情頁面
每顆行星都有獨立的詳情頁面，展示以下信息：
- **行星名稱**和**英文名稱**
- **直徑**: 行星的直徑（公里）
- **距太陽距離**: 行星與太陽的平均距離
- **衛星數量**: 環繞該行星的衛星個數
- **行星類型**: 類地行星、氣態巨行星或冰巨星
- **自轉周期**: 行星自轉一圈所需的時間
- **公轉周期**: 行星繞太陽公轉一圈所需的天數
- **密度**: 行星的平均密度
- **詳細描述**: 關於該行星的有趣事實

## 使用方法

### 在本地運行
1. 在瀏覽器中打開 `index.html` 文件
2. 點擊太陽系圖中的任何行星可查看其詳情
3. 在行星詳情頁面中，點擊"返回首頁"按鈕可返回

### NASA API 集成

項目中已預設了 NASA API 的調用結構。要使用真實的 NASA 數據：

1. 訪問 [NASA API 網站](https://api.nasa.gov/) 申請 API 密鑰
2. 在 `js/api.js` 文件中修改以下部分：
   ```javascript
   const NASA_API_KEY = 'YOUR_API_KEY_HERE';
   ```

當前項目使用本地備用數據，這確保了網站即使在沒有 API 密鑰或 API 不可用時也能正常工作。

## 樣式和設計

- **深色主題**: 適合宇宙和太空主題
- **反應式設計**: 在所有屏幕尺寸上都能正常顯示
- **動畫效果**: 平滑的過渡和懸停效果
- **無障礙設計**: 支持鍵盤導航和屏幕閱讀器

## 鍵盤快捷鍵

- **Esc**: 從行星詳情頁返回首頁
- **Ctrl/Cmd + Home**: 從任何頁面返回首頁
- **Tab**: 在交互式元素之間導航
